# Cgengo

> 中学・高校・大学時代に書いたC言語プログラム集

## 概要

中学・高校・大学の授業や自習で書いたC言語のプログラムをまとめたリポジトリです。
`kantan`（簡単）と`muzukasi`（難しい）の2ディレクトリに分けて管理しています。

- `kantan`：中学・高校時代に作成したプログラム（GPA計算、ソート、素数判定、最大公約数、うるう年判定など）
- `muzukasi`：大学時代に作成したプログラム（TCP通信、ネットワーク関連なども含む）

## ⚠️ 注意

このリポジトリは学習・練習目的で作成したものです。コードに誤りや古い書き方が含まれている可能性があります。動作は保証しません。

## 使用技術（わかる範囲で）

- C 言語
